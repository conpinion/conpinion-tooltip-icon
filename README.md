conpinion-tooltip-icon
======================

An icon with a tooltip.

Extends
-------

Attributes
----------

  * **description**
      - *type:* string
      - *default:* undefined
      
      The tooltip text
 
  * **icon**
      - *type:* string
      - *default:* 'help'
      
      The icon to show

  * **position**
      - *type:* string
      - *default:* 'bottom
      
      The tooltip position relative to the icon

  * **tooltipWidth**
      - *type:* string
      - *default:* 'auto'
      
      The tooltip width as a css
